# AuthTester

**Dependencies:**</br>
- paramiko</br>
- termcolor</br>

**Usage:**</br>
  -h, --help     Show this help message and exit</br>
  -t, --target   Target ip</br>
  -T, --targets  File with targets ip</br>
  -l, --login    Login</br>
  -p, --password Password</br>
  -f, --fast     Fast mode</br>
  -m, --mode     Protocol type (SSH/FTP)</br>
